# About Me 😃

My name is Timothy Enderson `(which is my online identity and not my real name)`, and I'm currently taking a look at Web Development, while also exploring various other areas, while I keep on discovering new ones 🌟.

I’ve been engaged with computers since I was young, which ignited my passion for technology.
Early on, I learned to analyze and solve problems through modding, server management, and debugging games.
My interest in design and visual effects led me to Adobe After Effects and Photoshop in 2015.
Around 2018, I came into contact with Linux through the Ubuntu server and loved figuring out how to make things work.
Initially, I started my education (apprenticeship) in system integration but soon found that software development suited my creative interests better.
This shift allowed me to uncover my enthusiasm for web development.

I have a solid foundation in web technologies, starting with the classic trio of HTML, CSS, and JavaScript.
My journey began with PHP 8+, where I explored it's object-oriented programming, while also learning SQL in a more practical manner, and the MVC pattern, before transitioning to JavaScript frameworks with Node.js.
I also have a good understanding of REST APIs.

My main focus is on full-stack web development, and I’m always eager to learn more in the IT field.
I’m passionate about many topics, which makes it feel like I’ve just scratched the surface 🌍.

I’ve learned both Java and C#, and I prefer C# between the two 🔥.
Although Java still has relevance to me, thanks to Minecraft, C# has offered a better developer experience.
Its explicit nature make it enjoyable to work with.
However, I'm trying to avoid both Java and C#, as there are more existing options.
