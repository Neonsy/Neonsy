# About Me 😃

Hey there! I'm Timothy Enderson (my digital alias), a passionate Full-Stack Web Developer with an insatiable curiosity for all things tech 👋

My tech journey started early - I was that kid who couldn't resist tinkering with computers, diving deep into game modding, and managing servers just for fun.
This hands-on experience taught me valuable problem-solving skills that still serve me today.

In 2015, I discovered my creative side through Adobe After Effects and Photoshop, which opened up a whole new world of digital possibilities.
By 2018, I was exploring the Linux ecosystem through Ubuntu server, embracing the challenge of making things work from scratch.
While I initially pursued system integration, my creative drive led me to discover my true calling in software development.

I've built a robust foundation in web development, mastering the essential HTML, CSS, and JavaScript stack.
My technical journey took an exciting turn with PHP 8+, where I delved into object-oriented programming and practical SQL applications.
I'm well-versed in the MVC pattern and have seamlessly transitioned to modern JavaScript frameworks with Node.js.
RESTful APIs? They're second nature to me now.

Full-stack web development is where I thrive, but I'm constantly expanding my horizons in the vast IT landscape.
I've explored both Java and C# (with C# winning my heart 🔥), and I'm amazed by how each new technology I learn reveals even more exciting possibilities to discover 🌍

I'm always on the lookout for new challenges and opportunities to grow.
Got an interesting project in mind?
Let's connect and create something awesome together! 💡
